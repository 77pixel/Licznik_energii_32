# Licznik energii
Licznik energi zbudowany w oparciu o ESP32 i przetwornik ADC ADS1115.

Jednoczesny pomiar zużycia energii elektrycznej dla 3 faz (prąd, natężenie), aplikacja WWW z wykresami w formie PWA.
